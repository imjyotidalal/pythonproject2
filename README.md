# 📚 AI-Based Study Habit Tracker

## 🧠 Overview

This is a simple Python-based AI Study Habit Tracker that helps students track their study sessions and get intelligent suggestions based on their personality type.

---

## ✨ Features

* Track daily study sessions
* Personality-based learning behavior
* AI-generated study suggestions
* Performance prediction system
* Simple menu-driven interface

---

## 👤 Personality Types

1. Consistent Learner
2. Last-Minute Crammer
3. Balanced Learner
4. Distracted Learner

Each personality gets customized AI feedback.

---

## 🤖 AI Logic

The system analyzes:

* Total study hours
* Number of study days
* Average study time

Then provides:

* Smart suggestions
* Performance prediction

---

## ▶️ How to Run

1. Install Python (if not already installed)
2. Save file as `study_tracker.py`
3. Run the program:

```
python study_tracker.py
```
